# Open Platform Definition

art.kubus is an open platform for artist discovery, map-based exploration, and community participation.

## What is open

- Open-source client code in this repository (Apache-2.0).
- Public platform API interfaces and documentation explicitly published as open.

## What is not open by default

- Backend implementation and production operations.
- Internal moderation, anti-abuse, and infrastructure systems.
- Brand assets, trademarks, and most content/media assets.

## API and ecosystem model

The project supports a public-platform model similar to community and map ecosystems:
- interoperable client features,
- public API integrations,
- user portability and transparent client behavior.

API usage is governed by Terms of Service and Developer API Terms.

## User-generated content (UGC)

Users retain ownership of the content they upload.
Platform operation may require limited rights to host, process, and display content under applicable terms.
