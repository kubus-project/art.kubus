# Screenshots

This repo keeps UI screenshots under `docs/screenshots/` so GitHub docs can reference real UI rather than mockups.

## Placeholders (add manually)

Add screenshots with these filenames:

| Screen | File |
|--------|------|
| Onboarding | `docs/screenshots/onboarding.png` |
| Home | `docs/screenshots/home.png` |
| Map (marker open) | `docs/screenshots/map.png` |
| Community | `docs/screenshots/community.png` |
| Artist Studio | `docs/screenshots/artist_studio.png` |
| Profile | `docs/screenshots/profile.png` |
| Institution Hub | `docs/screenshots/institution_hub.png` |

When you add them, you can reference them from Markdown like:

- `![Home](screenshots/home.png)`
