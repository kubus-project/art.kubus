Vector marker SVGs

This folder contains vector marker SVGs generated by the tool:

- `tool/export_map_marker_vector_svgs.dart`

How to generate (requires Dart/Flutter SDK available on PATH):

```bash
# from repo root
dart run tool/export_map_marker_vector_svgs.dart
```

The script downloads the 24px Material icon SVG sources from the
official Google `material-design-icons` repository and composes a simple
rounded-square marker SVG with the icon paths embedded. Output files are
written to this folder as `mk_<name>_vector.svg`.

Notes:
- The marker body uses a placeholder fill (#222222) so designers can
  recolor easily in Illustrator.
- If GitHub raw URLs change, the script will report fetch failures.
